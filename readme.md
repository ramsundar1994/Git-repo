Git New Repo Creation and Push to Origin Steps
1. Create One Repo name (folder) --> locally in system
2. Navigate to the folder and git init --> git init 
3. add the respective files and add to git --> git add .
4. commit the changes ---> git commit -m "readme file"
5. create repo in github with the same name (example: Git-repo) 
6. add the remote repo in local repo -->  git remote add origin https://github.com/ramsundar1994/Git-repo.git
7. edit the branch name (like Main) --> git branch -M main
8. need to push the changes to remote repo and it will create repo -->  git push --set-upstream origin main
